# 🎮 Simon Game

A classic Simon memory game built with vanilla JavaScript, HTML, and CSS. Test your memory skills by following increasingly complex color sequences!

## 🎯 About

Simon is a retro memory game that challenges players to repeat a growing sequence of colors. Each level adds a new color to the sequence, making it progressively harder. Can you reach the highest level?

## 📋 How to Play

1. **Start the Game**: Press any key on your keyboard to begin
2. **Watch the Sequence**: Simon will light up one button and play a sound
3. **Copy the Pattern**: Click the buttons in the exact same order Simon showed
4. **Advance Levels**: When you complete the sequence correctly, a new color is added
5. **Game Over**: If you press the wrong button, the game ends with a red flash
6. **Restart**: Press any key to play again

### Game Rules
- The game starts at Level 1 with 1 color
- Each level adds 1 more color to the sequence
- You must click the colors in the exact order shown
- One mistake ends the game
- Try to reach the highest level possible!
